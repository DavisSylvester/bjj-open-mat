## The operating principle
**"Automation" = a content engine + scheduler publishing from accounts you own** — never bots,
fake accounts, or scripted group spam. Instagram / TikTok / YouTube / FB-Page = scheduled.
**Reddit + FB Groups = manual, human, rules-checked** (automating them is a ToS + reputation risk).

## The weekly loop (the habit that matters most)
1. Batch-film clips from the shot briefs.
2. Load captions from `content-bank.md` into the scheduler per `content-calendar.md`.
3. Publish scheduled; engage live (Stories, Reddit, FB, comments).
4. Friday: pull analytics → clone winners, cut losers → adjust next week.

## What a human must do (can't be automated)
Reserve handles · stand up the waitlist page · **film clips** · hit publish / approve the queue ·
all community engagement · verify handle availability + benchmark claims before spending.
